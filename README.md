
# Dos Espacios - Test

## How to start

Clone the repository:

```sh
$ https://github.com/CindySalinas/dosespacios-test.git
```

Enter to cloned folder

```sh
$ cd dosespacios-test
```

Install the project's dependencies

```sh
$ npm install
$ bower install
```

Run server, watches your files

```sh
$ npm start
```

Run production mode

```sh
$ npm run build
```
